# app/entrypoints

## Назначение
Переиспользуемые helper-функции для CLI-точек входа.

## Ключевые файлы
- `common.py` — общий one-shot bootstrap:
  - загрузка/валидация конфига;
  - запуск раннера;
  - единый формат завершения и ошибок.

## Границы изменений
- Только CLI-обвязка и общие шаблоны запуска.
- Бизнес-логика обработки сообщений сюда не выносится.
