# certs

## Назначение
Локальное хранилище CA-сертификатов для Kafka TLS/SASL_SSL подключения.

## Ожидаемый файл
- `ca.crt` — корневой сертификат, который монтируется в контейнер как `/certs/ca.crt`.

## Важно
- Сертификаты локальные и обычно не коммитятся.
- Путь в env должен совпадать: `SSL_CAFILE=/certs/ca.crt`.
